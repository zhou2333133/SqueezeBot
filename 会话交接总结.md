## 1. 核心目标

构建一个**自学习、自进化但受硬边界约束**的加密货币合约超短线交易机器人。核心原则：

1. **交易核心用纯代码** — 信号生成、市场状态判断、风控、执行全部硬编码，不交给 LLM
2. **AI 不做交易决策** — LLM 只做：中文复盘报告、异常解释、辅助分析。不下单、不改参数、不改边界
3. **硬风控独立于 AI** — `risk_guard` 是执行前最后一道闸，`LOCKED_PARAMS` + `boundary_config` 双重锁定
4. **规则适配学习** — 系统记录"哪个候选来源 + 哪个币 + 哪种市场状态 + 哪套规则"的历史表现，自动规避亏损组合
5. **Evolver 只提案不越权** — 参数修改走 `proposal → risk_guard 审核 → 写入`，不能直接改 config

## 2. 已确认结论（架构冻结）

### 2.1 6 层架构（已冻结，不改）

```
第 1 层: 妖币扫描 → 选币（纯代码）
第 2 层: AI 判断 → 解释关注理由（LLM 辅助）
第 3 层: 规则学习器 → market_state → rule_selector → learning_memory（当前闭环完成）
第 4 层: Evolver → 产生 proposal，不直接写 config
第 5 层: risk_guard → 硬风控，执行前最后一道闸
第 6 层: execution_router → 只执行，不判断行情
```
